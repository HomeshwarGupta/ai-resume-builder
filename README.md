# AI Resume Builder

## Overview
AI-powered resume generator built using Flask and Groq API.

## Features
- Generates structured resume using AI
- Dynamic HTML output
- Secure API key handling
- Environment variable configuration

## Tech Stack
- Python
- Flask
- Groq API
- HTML/CSS

## Installation
1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create .env file and add:
   GROQ_API_KEY=your_api_key
4. Run:
   python app.py
